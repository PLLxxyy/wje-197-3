# 像素画编辑器

一个基于浏览器的像素画编辑器，支持多图层、撤销重做、导出 PNG 和本地作品管理。

## 功能特性

- **画布尺寸**：支持 16×16、32×32、64×64 三种网格大小
- **绘画工具**：画笔、橡皮擦、油漆填充、吸管取色、图层移动
- **多图层**：支持添加、删除、复制图层，调整图层透明度和顺序
- **缩放与网格**：1x–32x 缩放，可开关网格辅助线
- **撤销/重做**：最多 50 步历史记录，支持 Ctrl+Z / Ctrl+Y
- **导出 PNG**：将当前画布导出为高清 PNG 图片
- **作品管理**：保存到 localStorage，可打开历史作品继续编辑
- **调色板**：内置 30 色调色板 + 自定义颜色选择器

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| B | 画笔工具 |
| E | 橡皮擦工具 |
| F | 填充工具 |
| I | 吸管工具 |
| M | 移动工具 |
| [ / ] | 减小/增大笔刷尺寸 |
| Ctrl+Z | 撤销 |
| Ctrl+Y / Ctrl+Shift+Z | 重做 |

## 启动

```bash
npm install
npm run dev
```

浏览器会自动打开本地开发服务器（默认 http://localhost:5197）。

## 技术栈

- Vite + React 18 + TypeScript
- HTML Canvas API
- localStorage 持久化
- 无第三方 UI 库
